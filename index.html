An Incomplete
Guide to Basic HTML/CSS

(Specifically to build a website...)

HTML

Some Terms

Element Name Description
<span></span> Span Inline container for text and can be used to group text for styling purposes
<em></em> Emphasis Italics!
<ul></ul> Unordered List Bullet points
<ol></ol> Ordered List Numbers
<img> Image Empty element. Needs src=
alt= Alt Text Lives inside img element
<video></video> Yep. Needs src=, controls and "video not supported" text
<a></a> Link Needs an href=
target= Target Nested in <a>. Tells how link opens. Typically "_blank"

CSS

Some Other Terms

Element Description Example
font-family: Font "Helvetica", "Times New Roman", serif;
font-size: Font Size 12px;
font-weight: Font Weight bold;
text-align: Text Alignment center; right; left;
color: Foreground Color #22f; red;
background-color: Background Color aqua;
opacity: Transparency 0.5; (0=transparent, 1=opaque)
background-image: You guessed it. url("moon.jpg")
